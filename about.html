About This Project

This Interactive Learning Tool is a minor project developed for our 4th semester curriculum.

About the Creators

This project was created by Akashdeep Singh and Parshant Singh. We are friends and classmates pursuing a B.Tech in Computer Science & Engineering with a specialization in AI & Machine Learning. We are currently in our 4th semester.

The goal of this project was to create a hands-on, engaging platform to help students visualize and understand complex computer science concepts. Instead of static text and diagrams, this tool allows users to interact with algorithms and protocols directly, providing a more intuitive learning experience.

Thank you for checking out our work!
